How Does Our Machine Learn?

Our team of quants uses a machine learning model that provides investors with two main features: Stock Price Predictions and a Portfolio Optimizer.

Basic Flowchart

0

This stage will be the data collection process, and doing some preliminary data cleaning.
This will ensure that our data is structured the way we want and unnecessary data are cleaned out for faster calculations.

0

This stage is the feature extraction and selection process where we identify and choose which fundaamental information to use. These will be further elaborated below.

0

This is the stage where most processing power goes into. Our algorithm consists of 5 models: Linear Regression, Ridge Regression, Stepwise Regression, Random Forest and Generalized Boosted Regression.
Our algorithm will run all 5 models with the pre-processed data, together with the 20 features.

0

During this stage, we test each model's forecasted returns with the stocks' true returns. We will take note of the accuracy of each model using MSE, or mean-squared error. The higher the MSE, the lower the accuracy of the model.

0

Lastly, our algorithm adopts the model with the lowest MSE for each quarter. This is to ensure that we use the most accurate model for every stock forecast.

Fundamental Information Used (Features)

Our model uses these 20 fundamental information to forecast the next quarter returns of stocks. We select representative stock indicators with good explanatory power, and these 20 indicators allow us to understand the profitability, efficiency and level of leverage of a company.
